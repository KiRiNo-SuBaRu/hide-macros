# auto-surround_with_brackets_or_block-comment-delimiters

秀丸エディタ用マクロです。  
範囲選択した文字列または行を、選択した対括弧・ブロックコメント区切り記号・環境変数記法で囲みます。

## 作者

霧野昴 (Subaru Kirino)

##

## マクロの登録方法

1. このファイルを秀丸エディタのマクロ用フォルダーに配置します。
2. ［マクロ］→［マクロの登録］8個あるグループの中から、1つを選択します。  
   右側の［…］（詳細メニュー）ボタンをクリックすると「ファイル選択のダイアログ」が開きます。  
   そのダイアログのファイル一覧からこのファイルを選択し、［開く］をクリックします。
3. このマクロに分かりやすい名前を付けます。
   - 例：括弧自動補完
4. このマクロに割り当てるショートカットキーを登録します。

## 機能

- 範囲選択中のみ動作
- 非選択時は何もせず正常終了
- ポップアップメニューから囲み記号を選択（キーボードショートカット対応）
- 単一行選択、複数行選択に対応
- 複数行選択時は各行の前後に記号を挿入
- 処理後、最後に挿入した閉じ記号位置へカーソル移動

## 対応候補とキー割り当て

ポップアップメニュー表示中に下記のキーを押すと該当項目を選択できます。

| キー | 記号 | 説明 |
|:---:|---|---|
| `1` | `( )` | 半角丸括弧 |
| `2` | `[ ]` | 半角角括弧 |
| `3` | `{ }` | 半角波括弧 |
| `4` | `< >` | 半角山括弧 |
| `5` | `「 」` | 鉤括弧 |
| `6` | `『 』` | 二重鉤括弧 |
| `7` | `（ ）` | 全角丸括弧 |
| `8` | `【 】` | 全角隅付き括弧 |
| `E` | 編集中の拡張子で自動切替 | 環境変数形式（`.ps1` → `${env:VAR}` / それ以外 → `"%VAR%"`） |
| `C` | `/* */` | ブロックコメント |
| `D` | `/** */` | Docブロックコメント |
| `H` | `<!-- -->` | HTMLコメント |
| `T` | `<% %>` | テンプレートタグ |

## 想定用途

### Windows 環境変数

選択文字列:

```text
APPDATA
```

変換後（`.ps1` 以外のファイルを編集中）:

```text
"%APPDATA%"
```

### PowerShell 環境変数

選択文字列:

```text
APPDATA
```

変換後（`.ps1` ファイルを編集中）:

```text
${env:APPDATA}
```

## 使い方

1. テキストを範囲選択します。
2. マクロを実行します。
3. ポップアップメニューから囲みたい記号を選びます。
